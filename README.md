# Flux Med Website

Professional healthcare operations website for Flux Med - a trusted healthcare operations partner specializing in patient population management, risk stratification, and value-based care enablement.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Build for Production

```bash
npm run build
npm start
```
